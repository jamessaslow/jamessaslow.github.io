Project Description

This project is a report investigating the journal "Entropic Forces in Brownian Motion", by Nico Roos. In his paper, he talks about how the rms displacement of a system undergoing Brownian motion can be described as an Entropic Force evolving the system. This approach allows statistical mechanics problems to be solved from a different perspective. In the following paper, I compute a random walk via a Markov Chain problem and analyze the entropic forces at play.
