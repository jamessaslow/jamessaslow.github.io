Gaussian Mixture Model (GMM) in N dimensions

Required Skills for completing this project

  • Python Programming
  • GMM and K-means
  • Machine Learning
  • Classification Algorithms
  • Unsupervised Learning Methods
  • Statistics involving multivariate gaussians

Project Description

This project performs K-means followed by GMM on arbitrary data generated from multivariate Gaussians. The script detects the clusters and identifies their mean, covaraince matrix, and probability of assignment.
